# 国民の祝日


国民の祝日の一覧を取得します。

```js
import { nationalHolidays } from 'japanjs'

console.log(nationalHolidays)
// [
//   { 
//     name: '元日',
//     date: 2023-12-31T15:00:00.000Z
//   },
//   {
//     name: '成人の日',
//     date: 2024-01-07T15:00:00.000Z 
//   },
//   {
//     name: '建国記念の日',
//     date: 2024-02-10T15:00:00.000Z
//   },
//   ...
// ]

```
